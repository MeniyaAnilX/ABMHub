# ABMHub

ABMHub is a Next.js + Supabase web app.

## Local setup

1. Install packages:

```bash
npm install
```

2. Create `.env.local` from `.env.local.example`.

3. Run development server:

```bash
npm run dev
```

## Deploy

Deploy with Vercel and add the required environment variables in Vercel Project Settings.

Do not commit `.env.local`, private keys, tokens, or local config files.
