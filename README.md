ABMHub
=======
ABMHub is a Next.js + Supabase app.

## Setup

```bash
npm install
npm run dev
```

Create `.env.local` locally with:

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

Do not upload `.env.local`, config files, SQL files, or personal deploy BAT files to GitHub.
